# 🌱 Lanzi Orto Urbano — Frontend

Frontend of the Lanzi Orto Urbano e-commerce platform for fresh microgreens, built with **Next.js** and **Tailwind CSS**.

> 🚧 This project is currently under active development. Some features are available for preview; full functionality is planned for the next release.

---

## 🛠️ Technologies

| Technology | Usage |
|---|---|
| Next.js | React framework (App Router) |
| Tailwind CSS | Styling |

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
